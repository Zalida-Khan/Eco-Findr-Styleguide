Home

We selected a house icon for our app's homepage to evoke familiarity and comfort. Universally recognized as representing "home," it ensures easy navigation back to the main page. This choice aims to enhance user experience by providing a clear and intuitive way to access the main page

Pin

We used the pin icon in the navigation bar to signify location-related features, such as event searches on the map. Its familiar appearance makes it easy for users to recognize and associate with map-related actions. This choice simplifies navigation, ensuring users can quickly access location-based functionalities within our app.

Calendar

We incorporated a calendar icon in the navigation bar to indicate access to the upcoming event reminder calendar page. The calendar is universally recognized as a symbol for scheduling and planning, making it intuitive for users to identify where they can view and manage their upcoming events. This choice simplifies navigation, ensuring users can easily locate and utilize the calendar feature within our app to stay organized and informed about their upcoming events.

Profile

We included a person profile icon in the navigation bar to provide users with easy access to their personal profile and account settings. The icon represents individual user accounts, allowing users to view and manage their profile information, preferences, and account settings. By placing this icon in the navigation bar, we aim to streamline the user experience, enabling users to access their profile quickly and efficiently from any page within the app.

Search Icon

We incorporated the search icon in our web app to improve user experience by providing a universally recognized symbol for search functionality. Overall, it enhances accessibility and usability while maintaining a clean and visually appealing interface.

The bike icon on the map provides a clear indication to users that they can filter for biking events. This ensures relevance for users interested in biking activities and maintains consistency with widely recognized symbols. By offering a dedicated filter, users can efficiently find biking events without wasting time on unrelated listings, streamlining their experience within the app.

Hamburger Icon

The hamburger menu icon gives users an easy way to access app features. It's familiar and helps keep the interface clean. Users can quickly find what they need without clutter. We decided to stick with the standard design to keep it simple and recognizable.

Cancel Icon

The "X" symbol provides users with a simple way to delete their search input within the app. It offers a clear method to reset search parameters and start anew, contributing to a clean and intuitive interface. By incorporating the "X" symbol, users can easily manage their search queries without confusion, improving the overall usability of the search functionality.

Buttons

Get Started

The "Get Started" button allows users to bypass the tutorial and jump directly into using the app's features. This option offers convenience for those who prefer to explore the app independently, enhancing the user experience with flexibility.

Tutorial

We made the tutorial button green, contrasting with the "Get Started" button, to emphasize its purpose of guiding users through learning the app. This visual differentiation helps users easily distinguish between options: green for accessing tutorials and another color for diving straight into using the app.

Join

We utilized a "Join" button to enable users to participate in events. This straightforward label communicates the action clearly, indicating that users can easily join events they're interested in attending.

Create Event

After clicking the edit (pencil) button, we implemented a "Create Event" button that pops up, allowing users to seamlessly add events to their calendar. This design ensures a smooth and intuitive process for users to create new events without navigating away from the editing interface. By integrating this feature, we aim to enhance user efficiency and convenience when managing their calendar within the app.

Pencil icon

We chose a pencil icon button to represent editing and adding events to the calendar. This universal symbol for editing makes it intuitive for users to understand its function. By using the pencil icon, we aim to streamline the process of managing events, allowing users to easily make changes or add new events to their calendar with minimal effort.

Favorite

We selected a heart icon button to signify favouriting and saving events for later. The heart symbolizes affection or liking, making it intuitive for users to understand its function of saving events they're interested in. This choice simplifies the process of bookmarking events, enabling users to easily access and revisit their favourite events at any time with just a single click.
